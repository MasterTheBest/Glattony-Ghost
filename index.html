Glattony-ghost

A simple way to start with ghost, this amazing cms, with a simple template. With bootstrap 3 integrates, many features and a little pieces of "override", every page can be amazing. It's good for every ghost-lover.

Download this project as a .zip file Download this project as a tar.gz file

How to do?

It's the same for every ghost's template

1 Download

Ok, it's not so easy :)

2 Unzipped

Unzip inside a unique folder, and call it glattony

3 Install

Go to your ghost's folder installation and put entire folder inside 

root/content/themes/

4 Restart

Restart your ghost's installation

5 Select and Enjoy

Go to

www.youdomain.com/ghost-folder-installation/ghost

and log in it. Select menu voice settings and find theme field. Select glattony and save! Enjoy!!!
